# Keybindings

Complete list of keyboard shortcuts.

## Navigation

| Key | Action |
|:----|:-------|
| `j` / `↓` | Move down |
| `k` / `↑` | Move up |
| `gg` / `Home` | Go to top |
| `G` / `End` | Go to bottom |
| `Ctrl+d` | Half page down |
| `Ctrl+u` | Half page up |
| `g` | Jump to current worktree |
| `/` | Search worktrees |
| `Enter` | Enter worktree (cd) |

## Worktree Actions

| Key | Action |
|:----|:-------|
| `a` | Add new worktree |
| `d` | Delete worktree |
| `r` | Refresh list |
| `s` | Cycle sort mode |

## Git Operations

| Key | Action |
|:----|:-------|
| `f` | Fetch remotes |
| `p` | Pull from remote |
| `P` | Push to remote |
| `m` | Merge upstream |
| `M` | Merge branch (select) |

## External Apps

| Key | Action |
|:----|:-------|
| `o` | Open in editor |
| `t` | Open in terminal |

## Other

| Key | Action |
|:----|:-------|
| `y` | Copy path to clipboard |
| `c` | Open config modal |
| `?` | Show help |
| `q` | Quit |
| `Ctrl+c` | Quit |
| `Esc` | Close modal / clear filter |

## Add Worktree Modal

### Type Selection

| Key | Action |
|:----|:-------|
| `f` | Feature branch |
| `b` | Bugfix branch |
| `h` | Hotfix branch |
| `c` | Custom (no prefix) |
| `Esc` | Cancel |

### Branch Input

| Key | Action |
|:----|:-------|
| `Enter` | Create worktree |
| `Esc` | Back to type selection |
| `Shift+F` | Fetch base branch |
| `Shift+L` | Use local as base |
| `Shift+U` | Use remote as base |

## Delete Confirmation

| Key | Action |
|:----|:-------|
| `y` / `Enter` | Confirm delete |
| `n` / `Esc` | Cancel |
| `b` | Toggle delete branch |

## Config Modal

| Key | Action |
|:----|:-------|
| `j` / `k` | Navigate options |
| `Enter` | Edit selected option |
| `s` | Save config to file |
| `Esc` / `q` | Close config |

## Search Mode

| Key | Action |
|:----|:-------|
| (any text) | Filter worktrees |
| `Enter` | Enter selected worktree |
| `Esc` | Cancel search |
| `Backspace` | Delete character |
